# BrandUp

Landing page para a BrandUp, agencia de marketing focada em restaurantes na Australia.

## Stack

- React
- TypeScript
- Vite
- Montserrat
- Lucide React

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Observacoes

- O CNPJ no rodape esta como placeholder.
- Os links de redes sociais estao como placeholders e devem ser trocados pelos perfis reais.
